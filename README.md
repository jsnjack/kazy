kažy
====

### What is it?
kažy is an application that highlights output from STDIN

### How to install?
 - Clone this repository
 - Run
```bash
sudo python setup.py install
```

### How to use?
kažy is extremely useful when piping a command:
```bash
tail saturday | kazy sunny " " "###" 22 rain
```
